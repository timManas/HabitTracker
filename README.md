# Habit Tracker

Frontend: React <br/>
Backend: Node/Express <br/>
DB: MongoDB <br/>

## Purpose

Create HabitTracker which has following functionality:

1. Login new users and issue token <br/>
2. Verify JWT token <br/>
3. Once logged in, perform CRUD Operations <br/>
